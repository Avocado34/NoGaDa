# 주요 기능  

## 1️⃣ 부르고 싶은 노래를 검색해요! 실제 노래방에 등록되어있는 노래들을 바탕으로 결과를 보여줘요.
* 필터를 적용시켜서 원하는 노래를 빠르게 찾을 수 있어요.
* 부르고 싶은 노래를 폴더별로 정리해서 저장해 보아요!
* 등록되지 않는 노래는 직첩 추가해볼 수도 있어요!
